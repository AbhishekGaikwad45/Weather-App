# Weather-App


<img width="100%" alt="Weather App Screenshot" src="https://github.com/user-attachments/assets/c41d3438-6072-4ff5-b0f6-7bf9729aa878" />

A simple and responsive **Weather Application** built using **HTML, Tailwind CSS, and JavaScript**.  
This app fetches **real-time weather data directly from the OpenWeatherMap API** without using any backend.

---

## Features

-  Search weather by **city name**
-  Displays **current temperature**
-  Weather **description** (clear sky, clouds, rain, etc.)
-  **Humidity** level
-  **Sunrise & Sunset** time
-  Clean & modern UI using **Tailwind CSS**
-  Fast performance powered by **Vite**

---

##  Tech Stack

- **HTML5**
- **Tailwind CSS**
- **JavaScript (ES6)**
- **Vite**
- **OpenWeatherMap API**
